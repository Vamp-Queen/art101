Problems

I had the most trouble with resizing my selfie image. I had to use CSS and adjust the width, but for some reason it wouldn't change the image size. After troubleshooting with my lab partner, I discovered that I wasn't using the same class name between html and css. Once I changed them both to "selfie," the image resized as it should have. I also helped my partner to get their links working, which required double checking that the reference was correct (lab3/index.html).
